# TalentPulse AI Security Questionnaire

## Data processed
Employee names, employee emails, department, manager names, engagement survey responses, performance ratings, salary bands, and attrition-risk outputs.

## Integrations
HRIS and Slack.

## Security controls
- Encryption in transit: Yes
- Encryption at rest: Yes
- SSO/SAML: Yes
- SCIM provisioning: TBD
- Audit logs: Yes, but export API is not included in base package.

## SOC 2
SOC 2 Type I available. SOC 2 Type II: Not provided.

## Subprocessors
AWS (US), ModelOps Labs (EU), AnnotationWorks (APAC).

## Data retention
Default retention is the contract term plus 48 months. Custom deletion terms available by negotiation.

## AI/model training
Customer data may be used for model enhancement and benchmarking unless the customer purchases the Enterprise Control package and submits an opt-out request.
