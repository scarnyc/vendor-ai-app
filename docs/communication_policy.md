# Communication Policy

## Allowed agent actions

The agent may:

- Summarize the vendor request
- Identify missing information
- Draft internal follow-up messages
- Draft external vendor follow-up messages
- Recommend approval routing
- Create draft internal tickets
- Escalate unclear or risky cases to a human owner

## Prohibited autonomous actions

The agent must not:

- Approve a vendor
- Commit company spend
- Accept contract terms
- Send external communications without human approval
- Bypass Legal, Security, Finance, Procurement, or executive approval
- Modify contract language without Legal review
- Make final security or privacy decisions

## Human approval

All external communications require human approval.

All approval decisions require the responsible human approver.

The agent should clearly label generated messages as drafts.
