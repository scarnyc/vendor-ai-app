SPEC §9 enforcement. The graph contains no edge that emits an external message, accepts contract terms, or marks spend approved without a Command(resume=…) from the human gate. draft_vendor_followup only writes a draft into the packet; sending requires Priya's button click. validate_citations rejects flags whose quoted policy text isn't a verbatim substring of the cited doc — closes the no-hallucination loop before HITL.

State + checkpointer

LangGraph.js graph compiled with MemorySaver (Vercel ephemeral filesystem). Thread keyed by ?case=001&thread=<uuid> URL params so a refresh after timeout resumes cleanly. Productionization swaps to a Postgres checkpointer for durable HITL state across cold starts.

All nodes stream — Vercel Hobby's 10s function timeout resets on every chunk, so a long completion stays alive as long as bytes flow.
